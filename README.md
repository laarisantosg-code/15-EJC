# Sistema de Controle de Estoque

Este é um sistema web responsivo e moderno para a gestão e controle de estoque de mercadorias.

## Para que serve?

O projeto serve para facilitar o controle de inventário de produtos, oferecendo as seguintes funcionalidades:
- **Gestão de Inventário**: Cadastro, visualização, edição e exclusão de produtos em estoque.
- **Movimentações**: Ajuste rápido de saldo de estoque (+/-) e controle de saídas de mercadorias.
- **Relatórios**: Geração e exportação de relatórios completos em formato CSV, incluindo tabelas de inventário e histórico de saídas.
- **Painel Interativo**: Interface intuitiva para acompanhamento do status do estoque em tempo real.

## Linguagens e Tecnologias Utilizadas

O sistema foi desenvolvido utilizando as seguintes tecnologias:
- **HTML5**: Estruturação semântica de toda a aplicação e formulários.
- **CSS3**: Estilização personalizada, com suporte a temas (claro e escuro) e design responsivo.
- **JavaScript (ES6+)**: Lógica da aplicação, processamento de dados do estoque, geração de CSVs dinâmicos e controle de interface.
- **Supabase**: Integração com banco de dados em nuvem para persistência em tempo real das informações de estoque.
